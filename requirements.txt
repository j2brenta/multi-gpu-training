# torch 2.4/2.5 base images ("module 'torch' has no attribute 'int1'"). 0.4.0/0.7.0 is
# the validated combo for torch 2.5.1 (cu124) and has the qwen2_5_7b_base builder the
# config uses. setup_pod.sh ensures torch >= 2.5 before installing these.
